日本語UI対応の新着お知らせ管理プラグイン。カスタム投稿タイプとGutenbergブロックで、お知らせの一覧表示が簡単に行えます。

== Description ==

andW 新着お知らせ（andW Notices）は、日本語環境に最適化されたお知らせ管理プラグインです。

**主な機能**

* **カスタム投稿タイプ「notices」** - お知らせ専用の投稿タイプ
* **Gutenbergブロック** - ページや投稿にお知らせ一覧を簡単挿入
* **柔軟なリンク設定** - 自身のページ、内部ページ、外部URLへの対応
* **表示日の管理** - 公開日とは別に表示日を設定可能
* **管理画面カスタム** - 表示日、リンク先を一覧で確認
* **キャッシュ機能** - 高速な表示のための自動キャッシュ
* **完全日本語対応** - すべてのUIが日本語

**使用方法**

1. プラグインを有効化
2. 管理画面の「お知らせ」から新しいお知らせを作成
3. ブロックエディターで「お知らせ一覧」ブロックを挿入
4. ブロック設定で表示件数や並び順を調整

**ブロック設定項目**

* 表示件数（1-20件）
* 並び順（昇順・降順）
* 並び基準（表示日・公開日）
* 表示要素（日付・タイトル・抜粋の表示切り替え）
* リンク設定（新規タブ開く設定など）
* レイアウト（リスト・カード）

== Installation ==

1. WordPressの管理画面から「プラグイン」→「新規追加」
2. 「ANDW Notices」を検索してインストール
3. プラグインを有効化
4. 管理画面に「お知らせ」メニューが追加されます

**手動インストール**

1. プラグインファイルをダウンロード
2. `/wp-content/plugins/andw-notices/` ディレクトリにアップロード
3. WordPress管理画面でプラグインを有効化

== Frequently Asked Questions ==

= 表示日と公開日の違いは何ですか？ =

公開日はWordPressの標準機能ですが、表示日はお知らせ一覧での表示専用の日付です。
例えば、毎月1日に掲載したいお知らせを事前に作成し、表示日を1日に設定することで、
作成日に関係なく適切な順序で表示できます。

= 外部URLにリンクできますか？ =

はい。各お知らせで「外部URL」を設定すると、お知らせタイトルクリック時に外部サイトへ誘導できます。
セキュリティのため、設定でHTTPSのみに制限することも可能です。

= ブロックの表示をカスタマイズできますか？ =

ブロック設定で多くのオプションを調整できます。また、CSSで見た目をカスタマイズすることも可能です。
主要なCSSクラス：`.andw-notices`、`.andw-notice-item`、`.andw-notice-date`、`.andw-notice-title`

= キャッシュ機能について教えてください =

ブロックの表示内容は自動的にキャッシュされ、高速な表示を実現します。
お知らせの更新時にキャッシュは自動クリアされ、設定画面から手動クリアも可能です。

== Developer Notes ==

**技術仕様**

* WordPress 6.2以上対応
* PHP 7.4以上対応
* Gutenberg ブロックエディター対応
* REST API 対応
* WordPress Coding Standards 準拠

**フィルターフック**

* `andw_notices_allowed_url_protocols` - 許可するURLプロトコルの変更
* `andw_notices_cache_duration` - キャッシュ有効期間の変更

**アクションフック**

* `andw_notices_cache_cleared` - キャッシュクリア時に実行

**アンインストール**

プラグイン削除時に以下のデータが自動削除されます：
* お知らせ投稿データ
* メタデータ
* 設定オプション
* キャッシュデータ
